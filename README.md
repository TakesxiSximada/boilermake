# Boilermake


- common.mk

  Provide `make help` command, using `unmake` command.
  See: https://github.com/TakesxiSximada/unmake

- django.mk

  Provide `make django-*` command.

- docker.mk

  Provide `make docker-*` command.

- py.mk

  Provide `make py-*` command.

- doc.mk

  Provide `make doc-*` command.
